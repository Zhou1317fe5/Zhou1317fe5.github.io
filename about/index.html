关于我

我是一名对 机器学习、深度学习与人工智能 充满热情的学生，致力于探索 AI 技术在不同场景中的应用与优化。

学术背景

  • 完整学习了 Andrew Ng 的机器学习与深度学习课程，系统掌握了机器学习的基础理论与实践技能。
  • 熟练进行数据探索、预处理、特征工程、模型选择与融合等任务流程，并在 Linux 服务器上完整搭建过深度学习环境。

项目经历

  1. 私有化 AI 应用生态构建
    • 设计并实现了一个完整的私有化 AI 应用生态，包括底层 API 服务、AI 应用层和前端 Web 界面，服务于知识问答、代码解释等场景。
    • 支持多模型调用（如 Gemini、DeepSeek），并通过知识库集成提升了问答的准确性与实用性。
  2. 扩散模型在时间序列预测中的应用
    • 探索了 DDPMs 模型，以及Diffusion-TS扩散模型在时间序列预测中的应用。
  3. 数字孪生智能控制系统开发
    • 利用 Vuforia 搭建 AR 模型，并通过TCP 通信实现了数字孪生智能控制系统的开发。
  4. 创新创业项目
    • 参与多项创新创业项目（如挑战杯、三创赛），积累了丰富的团队协作与项目管理经验。
